# API Básica con FastAPI

API simple para practicar con FastAPI y GitHub.

## Instalación

```bash
#1 Instalar todas las dependencias
pip install -r requirements.txt

#2 ir a al carpeta del proyecto
cd "C:\Users\user\Documents\ProyectDocuments\VisualStudio\Control de Versiones"


# 3. Inicializar Git correctamente
git init

# 4. Agregar archivos
git add .

# 5. Commit
git commit -m "feat: primera versión de la API con FastAPI"

# 6. Configurar rama main
git branch -M main

# 7. Conectar con GitHub
git remote add origin https://github.com/GeovanniOrtiz/Test.git

# 8. Subir
git push -u origin main
